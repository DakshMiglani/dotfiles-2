Install Git:
    git

Enable AUR:
    /etc/pacman.conf file and add these lines at the bottom:
    
    [archlinuxfr]
    SigLevel = Never
    Server = http://repo.archlinux.fr/$arch
    
    Update the mirrors:
        sudo pacman -Syy

Install yay:
    git clone <yay repo>
    cd yay
    makepkg -si
    
    Update the mirrors:
        sudo yay -Syy
